# BabyRelay Shared-Care Screenshot Sequence V3

Generated for review on 2026-06-16 using the OpenAI Images API with
`gpt-image-2`.

These are review artifacts only. They have not been pushed to App Store
Connect or AppStoreCopilot.

## Goal

This pass responds to the design critique of the previous shared-care set:
the first frames were polished but too static and too visually similar. V3
keeps the approved warm dawn/night BabyRelay style while adding:

- distinct screen states from frame to frame
- truthful UI pop-outs based on visible BabyRelay components
- caregiver portrait/dotted-line supporting imagery
- stronger handoff and share-action proof moments

## Target

- Display target: iPhone 6.9-inch App Store screenshots
- Generated size: 1328x2880
- Final cropped size: 1320x2868
- Provider/model: OpenAI Images API, `gpt-image-2`

## Sequence

1. `One shared log for every caregiver.`
   `Parents, partners, grandparents, and nannies see the same day.`
   Hero mechanic: child-chip row pop-out plus caregiver portrait medallions.
2. `Log once. Everyone is updated.`
   `Sleep, feeds, diapers, notes, and wakes stay in one shared timeline.`
   Hero mechanic: floating timeline row with caregiver dots.
3. `No more handoff guesswork.`
   `See the last nap, feed, change, and note before you take over.`
   Hero mechanic: handoff summary card breaking out of the phone.
4. `Every child stays separate.`
   `Switch siblings without mixing naps, feeds, or notes.`
   Hero mechanic: split sibling timeline ribbons and active child-chip pop-out.
5. `Send the day in seconds.`
   `A ready-to-share summary turns care history into context.`
   Hero mechanic: share handoff button pop-out with receive cue.
6. `Build your baby care team.`
   `Invite helpers so every update stays shared.`
   Hero mechanic: caregiver portrait web and invite button pop-out.
