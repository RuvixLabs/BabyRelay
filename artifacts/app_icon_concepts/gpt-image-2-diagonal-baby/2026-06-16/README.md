# BabyRelay App Icon Concepts - Diagonal Baby Pass

Generated with `gpt-image-2` on 2026-06-16 after Joe asked for a concept with
just a baby placed diagonally.

## Concepts

1. `flat_diagonal` - cleanest flat/product mark.
2. `soft_clay_diagonal` - warmest literal baby-care read.
3. `paper_cut_diagonal` - refined layered cut-paper treatment.
4. `moon_swaddle` - most ownable/brandable diagonal baby-moon mark.
5. `glow_diagonal` - warm night/sleep personality, more literal.
6. `minimal_capsule` - simplest capsule-style launcher icon.

## Initial Recommendation

`01_flat_diagonal` and `06_minimal_capsule` are the cleanest true launcher
marks. `04_moon_swaddle` is the most distinctive BabyRelay-style mark if we
want the icon to feel more ownable than generic baby care. `02_soft_clay` and
`05_glow_diagonal` feel warmer but more literal.

No live app icon files were changed in `ios/Runner/Assets.xcassets`.
