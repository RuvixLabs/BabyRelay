Generated with `gpt-image-2` on 2026-06-16 after Joe asked for concepts with
two hands on either side and a baby in the middle.

## Concepts

1. `cupped_hands` - clearest literal safe-hands baby mark.
2. `side_hands` - explicit two-caregiver/shared-care symbol.
3. `night_cradle` - more polished protective cradle, slightly more detailed.
4. `shared_care_ring` - ring-like shared-care motion around the baby.
5. `handoff_baby` - handoff/pass-the-baby metaphor, simple and friendly.
6. `protective_parentheses` - simplest icon-safe version of the idea.

## Initial Recommendation

Use `06_protective_parentheses` for the cleanest launcher icon. Use
`01_cupped_hands` if the app icon should instantly read as baby care. Use
`02_side_hands` or `05_handoff_baby` if the shared-care/two-caregiver idea is
more important than maximum simplicity.
